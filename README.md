## Instalación y Uso

1. **Clonar el repositorio**:

   ```bash
   git clone https://github.com/Gian1611/appProyecto.git

   ```

2. **Instalar dependencias: Asegúrate de tener Node.js instalado y ejecuta:**:

   ```bash
   npm install

   ```

3. **Iniciar proyecto**:
   ```bash
   npm start
   ```
